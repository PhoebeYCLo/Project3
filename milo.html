MILO. is a fictional e-commerce site dedicated to selling modern furniture.

CONTEXT

This class project was about designing for an imaginary e-commerce site. In this project, I worked in team of 2. In our case, the goal was to create a site selling modern furniture.

STYLE GUIDE

First stage of the project, is to come up an idea and brainstorm for design for the site. After the brainstorm stage, me and my groupmate decided to work on a furniture website. We wanted to focus on creating a site that look modern and friendly. Therefore, we created a style guide that show the theme and overall style of the company.

BUILDING THE SITE

I was in charge of images selection, building the pages of home, shop, and shopping cart. I also using css to style some elements/ overall website layout (buttons, table chart etc) and make the site responsive.

I used Sublime text editor to code out the HTML and CSS in conjunction with Gitkraken to share files between myself and my partner.
